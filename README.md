#📊 Workforce Performance & Employee Insights Dashboard

##🚀 Overview

An interactive Power BI dashboard designed to analyze workforce performance, employee distribution, and training effectiveness.
It enables data-driven HR decisions by uncovering trends in engagement, performance, and attrition.

🎯 Key Highlights
📌 2845 Employees analyzed across departments, states, and demographics
📉 13.6% Termination Rate with department-level breakdown
📊 Performance Insights: Majority employees meet expectations, with clear improvement segments
🎯 Training Analysis: Tracks success rate, outcomes, and cost (1.59M)
😊 Engagement & Satisfaction: Identifies scope for workforce improvement
📊 Dashboard Insights
Workforce Distribution → Diversity & employee segmentation
Department Analysis → Production dominates workforce & attrition
Performance Metrics → Highlights high performers vs improvement areas
Training Outcomes → Completion, failure, and ROI tracking
Geographical View → State-wise workforce concentration
Termination Analysis → Department & gender-based patterns
